üîß Technical Issues

Why isn't my progress saving?

Game progress is stored in your browser's local storage. If your progress isn't saving, check these common causes:

  • Private/Incognito mode ‚Äî Data is cleared when you close the window
  • Cleared browser data ‚Äî Clearing cookies/cache removes game data
  • Different browser/device ‚Äî Data doesn't sync across browsers
  • Storage disabled ‚Äî Some privacy extensions block localStorage

For reliable progress saving, play in normal browsing mode without aggressive privacy settings.

The game isn't loading. What should I do?

If a game isn't loading properly, try these steps:

  • Refresh the page (Ctrl+R or Cmd+R)
  • Clear your browser cache
  • Disable browser extensions temporarily
  • Try a different browser (Chrome, Firefox, Safari, Edge)
  • Check your internet connection

If the problem persists, please use the Feedback button to report the issue!

Why isn't the audio playing in Blind Test?

Modern browsers require user interaction before playing audio. Make sure to:

  • Click the play button to start the audio sample
  • Check that your device volume is turned up
  • Ensure the browser tab isn't muted
  • If on mobile, make sure your phone isn't in silent mode
