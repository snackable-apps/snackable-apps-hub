# Movies Quiz Data

The Movies Quiz uses a curated database of films stored in Supabase.

## Data Properties

- Title, Director
- Genres (array)
- Release Year, Runtime
- IMDB Rating
- Country of Origin
- Cast (top 10 actors with images)
- Movie poster image
- Difficulty (easy/medium/hard)

## Data Management

Data import and management scripts have been moved to the private `snackable-api` repository for security (they contain API key references).

See: `snackable-api/scripts/movies/`
