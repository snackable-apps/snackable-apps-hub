📖 How to Play Clubs Quiz

Clubs Quiz is a daily guessing game for football fans. Your mission is to identify the secret football club using clues.

🏁 The Rules

  1. Make a guess — Type any football club name and select from the dropdown.
  2. Read the feedback — Each attribute shows how your guess compares to the secret club.
  3. Narrow it down — Use the clues to eliminate possibilities and find the answer.

🎯 Understanding the Clues

  • ✅ Green (Match) — This attribute matches exactly.
  • 🔼 Higher — The secret club has a higher value (older, bigger stadium).
  • 🔽 Lower — The secret answer has a lower value.
  • ❌ Red (Different) — The attribute doesn't match (different country, title, etc.).

📊 Available Clues

Each guess reveals information about:

  • Country — The club's country
  • Founded — The year the club was established
  • Stadium Capacity — How many fans the stadium holds
  • Titles — Major trophies the club has won
💡 Pro tip: Start with a well-known club to quickly narrow down the country and title range!
